# Housing Price Predictor

This project builds a machine learning pipeline to predict house sale prices
using the Ames Housing dataset.

## Dataset
- Source: Kaggle â€“ House Prices: Advanced Regression Techniques
- Target: SalePrice

## Approach
- Train-test split (80/20)
- Numeric preprocessing: median imputation + scaling
- Categorical preprocessing: most-frequent imputation + one-hot encoding
- Model: Linear Regression
- Evaluation metric: RMSE

## Results
- Test RMSE: ~29476
